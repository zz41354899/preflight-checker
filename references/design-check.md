# Design Check

Use for design drafts, banners, covers, thumbnails, posters, landing visuals, campaign assets, and visual material before sharing or publishing.

## Inspect For

- goal: what action or impression the design should create
- hierarchy: main message, secondary message, CTA, brand signal
- readability: text size, contrast, line breaks, cropping, mobile fit
- composition: alignment, spacing, balance, focal point, visual clutter
- asset quality: resolution, export format, safe area, compression, transparent background
- consistency: colors, typography, logo usage, visual system, tone
- channel fit: platform size, preview crop, file weight, dark/light background
- risk: misleading visual, missing disclaimer, wrong date/name/logo

## Common Risks

- headline not readable at thumbnail size
- CTA hidden or competing with decoration
- important text near crop edge
- image looks generic and does not show the real subject
- visual style does not match the audience or offer
- date, speaker name, logo, or title inconsistent with copy

## Output Notes

When no image is available, inspect the design brief or intended layout. Use the comparison table to map "current visual decision" to "recommended adjustment".
